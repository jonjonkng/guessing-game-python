# 🎮 Jogo de Adivinhação em Python

Um jogo interativo desenvolvido em Python onde o jogador precisa adivinhar um número secreto.

## 🚀 Funcionalidades

- 🎚️ Níveis de dificuldade (Fácil, Médio, Difícil)
- 💣 Limite de tentativas
- 🏆 Sistema de recorde
- 🔥 Dicas de proximidade
- 🔁 Opção de jogar novamente

## 🛠️ Tecnologias utilizadas

- Python

## ▶️ Como executar

```bash
python jogo.py
